Please find below the links to the software that I found helpful in my research.

Scientific python software

  • Anaconda python, ready to use distribution of python with scientific stack, including modules:
    SciPy and NumPy (libraries for scientific computations), 
    matplotlib (library for plotting), 
    pandas (powerful data analysis tool), 
    SymPy (symbolic computations),
    IPython (interactive computing),
    spyder (MATLAB-like IDE).

  • PyDSTool: dynamical systems analysis, numerical continuation.
  • PyGMO: a very powerful library for parallel multiobjective optimization developed by the European Space Agency.
  • OpenOpt: a package for numerical optimization.
